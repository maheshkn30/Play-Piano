# Play Piano 🎹

A simple interactive piano built using **HTML**, **CSS**, and **JavaScript**. This project allows users to play piano keys clicking on the piano keys using their mouse.

## Demo Link

https://maheshkn30.github.io/Play-Piano

## Features

- 🎵 Interactive piano with 2 different sounds
- 🎧 Play sounds when keys are pressed (mouse)
- 🌟 Designed using pure HTML, CSS, and JavaScript

## Tech Stack

**Front End:** HTML, CSS and JavaScript
